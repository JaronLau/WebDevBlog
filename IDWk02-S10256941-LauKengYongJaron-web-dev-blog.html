    Forms - Positioning - Media Queries - Flex

    Forms

    A HTML form is to collect user input and send it to a server for processing.
    The form element acts as a container for the input elements to collect different user inputs.
    Typees of input elements include
    • text
    • radio
    • checkbox
    • submit
    • button
    Labels are to show what the forms are for
    Label for 'fname' = First name:,
    Input type='text' id='fname' name='fname'



    Positioning

    Positioning property changes the positioning scheme of the element

    Types of positioning include:
    • Relative position
      • moves element relative to where it would be on the page
    • Absolute position
      • Elements are fixed in relation to the parent content
      • The offset is relative to the browser window
    • Fixed position
      • Element coordinate is always relativce to the entire browser window
      • elements do not scroll with page; sticky element
    • z-index
      • Controlls the depth of the element on the page
      • Elements with higher z-index overlapps those with lower value

    Media Queries

    The @media rule allowed for different styles according to different media types

    E.g. Phone screens can have different layouts and color schemes compared to the website on a desktop.
    • @media screen and (min-width:400px){styles to be added here e.g. body{bg:green}}

    Flex

    Flexbox container is the parent of the flex element;children.

    Children can be positioned and and flexible according to code and user needs
  • children can be positioned horizontally and vertically
  • children can grow and shrink according to window size
